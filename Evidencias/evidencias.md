# 📚 Aprendizajes

## 1. ¿Qué fue lo más fácil y lo más retador del proyecto?

Lo más fácil fue construir la estructura base en HTML, ya que las etiquetas semánticas permiten organizar el contenido de forma clara y lógica desde el inicio.

Lo más retador fue lograr que el diseño fuera realmente responsivo. Ajustar los elementos para que se vieran bien en diferentes tamaños de pantalla implicó modificar estilos, reorganizar secciones con Flexbox y probar varias veces hasta evitar problemas como desbordamientos o mala distribución del contenido.

---

## 2. ¿Qué partes de HTML semántico y Flexbox usaste y por qué?

Utilicé etiquetas semánticas como header, nav, main, section, article y footer para estructurar la página de forma clara, mejorar la accesibilidad y facilitar la comprensión del contenido tanto para usuarios como para navegadores.

En cuanto a Flexbox, lo usé principalmente en la navbar para alinear el logo y los enlaces, en la sección de cards para distribuir los elementos de forma flexible, y en algunas secciones para centrar contenido. Esto me permitió tener un control más dinámico del layout sin necesidad de usar medidas rígidas.

---

## 3. ¿Cómo organizaste tus media queries y breakpoints?

Trabajé con un enfoque mobile-first, comenzando con estilos base pensados para pantallas pequeñas y después agregando media queries para pantallas más grandes.

Utilicé un breakpoint principal alrededor de 768px para adaptar el diseño a tablet y escritorio. En este punto ajusté la dirección de los elementos en Flexbox (por ejemplo, pasar de columnas a filas) y mejoré la distribución del espacio para aprovechar mejor el ancho disponible.

---

## 4. ¿Qué mejorarías en una siguiente versión?

Me gustaría mejorar el diseño visual haciéndolo más atractivo, por ejemplo agregando una paleta de colores más trabajada y mejores tipografías. También implementaría un menú hamburguesa para móvil y pequeñas animaciones con CSS para mejorar la experiencia del usuario.

Además, optimizaría las imágenes y añadiría más secciones o contenido dinámico para hacer la página más completa y profesional.
